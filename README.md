## 解决方案
1.先建立一张 Hbase 表。

2.使用爬虫程序获取天气数据。

3.使用 kafak 实时分发数据。

4.利用 Flume 收集数据并写入到 Hbase 中。

5.创建 Hive 表与 Hbase 表进行关联。

6.使用 Superset 统计和展示 Hive 中的表。
